# Tongfengguan.github.io 个人站点

这是一个基于 Vue 3 + Vite 的个人站点，包含作品、链接、CF 数据展示等内容。portal 页使用了动态主题切换、粒子特效与实时时间显示。

## 快速开始

- 安装依赖: `npm install`
- 启动开发服务器: `npm run dev`
- 构建产物并输出到 docs/ 以用于 GitHub Pages: `npm run build`
- 预览本地构建产物: `npm run preview`
- 检查代码格式: `npm run format`
- lint/修复代码风格: `npm run lint`

> Node 版本建议：20.x 及以上。该项目在 engines 字段中指定了 Node 版本范围，请确保使用符合要求的版本。

## 目录结构要点

- docs/ 输出构建产物
- public/ 静态资源
- src/ 源代码（Vue 组件、路由、样式等）
- .github/ CI/工作流配置

## 主要依赖

- Vue 3
- Vue Router
- Pinia
- Vite
